Avant de contacter le support

  • Vérifier la version installée d’AraE
  • Consulter le changelog
  • Identifier le moment exact où le problème apparaît
  • Noter les étapes précises de reproduction

Sauvegarde recommandée

  • Conserver une copie régulière du dossier de données
  • Ne pas attendre une mise à jour importante pour faire une sauvegarde
  • Garder au moins une copie hors du poste principal
  • Vérifier qu’une sauvegarde récente reste exploitable

Informations utiles au support

  • Version d’AraE
  • Version de Windows
  • Étapes de reproduction
  • Résultat attendu vs résultat observé
  • Impact réel sur l’usage
Cas fréquents

Les demandes de support les plus probables

Une bonne page support ne se contente pas de donner un email. Elle aide l’utilisateur à qualifier rapidement son problème.

Problème d’installation

Si l’installation échoue, préparez la version de Windows utilisée, le moment exact de l’échec et tout message affiché à l’écran.

Comportement inattendu

Si une action ne produit pas le résultat attendu, décrivez précisément l’opération réalisée, le contexte et ce qui aurait dû se passer.

Données ou sauvegarde

Si le problème touche une campagne ou un dossier de données, sécurisez d’abord une copie avant toute manipulation supplémentaire.

Questions fréquentes

Où AraE stocke-t-il ses données ?
Le positionnement du produit reste local et offline-first. Cette page doit documenter le chemin exact de stockage dès que celui-ci est stabilisé côté application.
Comment restaurer mes données après réinstallation ?
La procédure de restauration doit être documentée ici dès que le format de données et le chemin de stockage sont définitivement fixés.
Comment signaler un bug proprement ?
Envoyez la version d’AraE, la version de Windows, les étapes de reproduction, le résultat attendu, le résultat observé et l’impact concret sur votre usage.
Quand faut-il faire une sauvegarde ?
Avant une réinstallation, avant une manipulation risquée, avant une mise à jour importante et plus généralement de manière régulière si votre campagne contient déjà des données utiles.
Comment contacter le support ?
Le point d’entrée principal est l’email contact@example.com.
Besoin d’aide

Préparez le contexte, puis contactez le support avec des informations exploitables.

Une demande claire accélère le diagnostic, réduit les allers-retours et améliore la qualité de la réponse.
