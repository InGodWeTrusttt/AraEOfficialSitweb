Avant d’ouvrir un ticket

  • Vérifier la version installée
  • Consulter le changelog
  • Décrire précisément le contexte de reproduction

Sauvegarde recommandée

  • Conserver une copie régulière du dossier de données
  • Ne pas attendre une mise à jour majeure pour faire un backup
  • Garder au moins une sauvegarde hors du poste principal

Informations utiles au support

  • Version d’AraE
  • Version de Windows
  • Étapes de reproduction
  • Impact réel sur l’usage

Questions fréquentes

Où AraE stocke-t-il ses données ?
Le positionnement du produit est local / offline-first. Complète ici avec le chemin réel de stockage quand il sera stabilisé côté application.
Comment restaurer mes données après réinstallation ?
Cette page est prévue pour documenter la procédure réelle de restauration. Ajoute-la dès que le format et le chemin de données sont figés.
Comment contacter l’équipe ?
Le point de contact principal du site est l’email contact@example.com. Remplace cette adresse dans la configuration avant publication.
Comment signaler un bug ?
Fournis le contexte, la version d’AraE, les étapes pour reproduire et l’impact observé. Une page support crédible réduit le bruit et améliore la qualité des retours.
